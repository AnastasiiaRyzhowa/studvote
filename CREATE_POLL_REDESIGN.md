# Редизайн страницы создания опроса

## Обзор изменений

Страница создания опроса была полностью переработана с использованием **Tailwind CSS** вместо Material-UI для создания более современного и чистого интерфейса.

## Основные изменения

### 1. Технологический стек
- ✅ Установлен и настроен **Tailwind CSS**
- ✅ Добавлен плагин **@tailwindcss/forms** для красивых форм
- ✅ Подключены **Google Fonts (Inter)** и **Material Symbols**
- ✅ Настроена кастомная цветовая палитра

### 2. Дизайн и UX

#### Цветовая схема
```javascript
{
  primary: '#0167FC',      // Основной синий цвет
  secondary: '#6366f1',    // Индиго
  accent: '#f59e0b',       // Янтарный/желтый
  danger: '#ef4444',       // Красный
  success: '#10b981',      // Зеленый
}
```

#### Структура страницы
- **Двухколоночный макет**: основной контент слева, настройки справа (на больших экранах)
- **Адаптивный дизайн**: на мобильных устройствах колонки располагаются вертикально
- **Карточный интерфейс**: каждая секция оформлена как отдельная карточка с иконками

### 3. Основные секции

#### Заголовок страницы
- Название и описание страницы
- Кнопка "Создать с AI" в правом верхнем углу
- Градиентный фон и эффекты при наведении

#### Основная информация
- Поле для названия опроса (обязательное, 5-200 символов)
- Поле для описания (опциональное, до 2000 символов)
- Кнопка генерации с AI
- Кнопка использования шаблона (только для преподавателей)

#### Варианты ответов
- Динамический список вариантов с буквенными метками (A, B, C...)
- Кнопка удаления варианта (появляется при наведении)
- Кнопка добавления нового варианта (до 20 вариантов)
- Минимум 2 варианта обязательны

#### Параметры опроса
- **Тип голосования**: один вариант / несколько вариантов / оценка
- **Максимум вариантов**: для типа "несколько вариантов"
- **Видимость**: все студенты / моя группа / мой факультет
- **Показ результатов**: сразу / после завершения / никогда
- **Анонимность**: переключатель (toggle)

#### Сроки
- **Начало**: опубликовать сейчас / запланировать
- **Длительность**: 1 день / 3 дня / неделя / свой срок
- Автоматический расчет даты окончания

#### Кнопки действий
- **Опубликовать**: основная кнопка с градиентом и анимацией
- **Сохранить черновик**: вторичная кнопка

### 4. Модальные окна

#### AI генерация
- Поле для ввода темы опроса
- Примеры подсказок
- Индикатор загрузки при генерации
- Обработка ошибок

#### Шаблоны (для преподавателей)
- Сетка с 6 готовыми шаблонами:
  1. Замена преподавателя
  2. Выбор элективных курсов
  3. Обратная связь по курсу
  4. Расписание лабораторных
  5. Выбор даты экзамена
  6. Темы курсовых проектов
- Каждый шаблон с иконкой и описанием
- Hover-эффекты

### 5. Интерактивные элементы

#### Анимации и эффекты
- Плавные переходы при наведении
- Масштабирование кнопок при нажатии
- Анимация загрузки (spinner)
- Glow-эффект для основных кнопок
- Появление кнопки удаления при наведении на вариант

#### Валидация
- Проверка минимальной длины названия (5 символов)
- Проверка максимальной длины (200 символов)
- Минимум 2 варианта ответа
- Проверка дат (окончание > начала)
- Отображение ошибок в красных карточках

### 6. Адаптивность

#### Брейкпоинты
- **Mobile**: < 768px - одна колонка
- **Tablet**: 768px - 1024px - адаптивная сетка
- **Desktop**: > 1024px - две колонки

#### Особенности
- Sticky-позиционирование кнопок действий на десктопе
- Адаптивные отступы и размеры шрифтов
- Скрытие некоторых элементов на мобильных устройствах

## Технические детали

### Конфигурация Tailwind

```javascript
// tailwind.config.js
module.exports = {
  content: ["./src/**/*.{js,jsx,ts,tsx}"],
  darkMode: 'class',
  theme: {
    extend: {
      colors: { /* кастомные цвета */ },
      fontFamily: { sans: ['Inter', 'system-ui', 'sans-serif'] },
      borderRadius: { /* кастомные радиусы */ },
      boxShadow: { 
        'soft': '...',
        'glow': '...'
      }
    },
  },
  plugins: [require('@tailwindcss/forms')],
}
```

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined...');

@tailwind base;
@tailwind components;

```
