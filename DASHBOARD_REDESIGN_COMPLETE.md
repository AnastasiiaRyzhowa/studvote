## Технические детали

### Цветовая схема

**Светлая тема:**
- Background: `#f3f4f6`
- Surface: `#ffffff`
- Text primary: `#111827`
- Text secondary: `#6b7280`

**Темная тема:**
- Background: `#0f172a`
- Surface: `#1e293b`
- Text primary: `#f8fafc`
- Text secondary: `#94a3b8`

### Акцентные цвета
- Primary Blue: `#0167FC`
- Purple/Sky: `#0ea5e9`
- Green: `#10b981`
- Red: `#ef4444`
- Orange: `#f97316`
- Secondary (Gold): `#f59e0b`

### Material Icons Round

Все иконки теперь используют Material Icons Round через CSS:

```jsx
<span className="material-icons-round">icon_name</span>
```

**Используемые иконки:**
- `school` - логотип
- `home`, `poll`, `add_circle`, `description`, `person`, `emoji_events` - навигация
- `search` - поиск
- `event_note` - календарь
- `group`, `event` - метаданные опросов
- `more_horiz` - меню
- `add` - создание опроса
- `light_mode` - переключатель темы
- `workspace_premium` - награды

## Особенности

### Адаптивность
- Правая панель скрывается на экранах < 1280px (lg breakpoint)
- Сетка опросов становится одноколоночной на мобильных
- Все компоненты адаптированы под разные размеры экранов

### Производительность
- Использованы useMemo для оптимизации рендеринга
- Плавные CSS transitions вместо JavaScript анимаций
- Минимизировано использование тяжелых Material-UI компонентов

### Доступность
- Правильные семантические теги
- Поддержка focus-visible
- Читаемые цвета с достаточным контрастом
- Hover-эффекты для всех интерактивных элементов

## Как запустить

```bash
cd client
npm install
npm start
```

Приложение откроется на `http://localhost:3000`

## Переключение темы

Нажмите на кнопку "Переключить тему (Demo)" в нижней части сайдбара для переключения между светлой и темной темой.

## Проблемы и решения

### React Calendar удален
**Причина:** Не соответствовал новому дизайну
**Решение:** Создан кастомный компонент календаря с полным контролем стилей

### Material-UI минимизирован
**Причина:** Конфликт стилей с новым дизайном
**Решение:** Использованы базовые Box компоненты и чистый CSS/Tailwind

### Темная тема
**Причина:** Необходима поддержка для Tailwind
**Решение:** Добавлен класс 'dark' в documentElement через ThemeContext
