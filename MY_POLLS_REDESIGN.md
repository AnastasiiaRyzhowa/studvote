# Редизайн страницы "Мои опросы"

## Обзор изменений

Страница "Мои опросы" была полностью переработана с использованием **Tailwind CSS** вместо Material-UI для создания более современного и чистого интерфейса.

## Основные изменения

### 1. Технологический стек
- ✅ Использование **Tailwind CSS** для стилизации
- ✅ Использование **Material Symbols** для иконок
- ✅ Сохранена интеграция с QRCodeDialog
- ✅ Адаптивный дизайн для всех устройств

### 2. Дизайн и UX

#### Цветовая схема
```javascript
{
  primary: '#0167FC',      // Основной синий цвет
  success: '#10b981',      // Зеленый для активных опросов
  warning: '#f97316',      // Оранжевый для акцентов
  danger: '#ef4444',       // Красный для срочных дедлайнов
  gray: различные оттенки  // Для текста и фонов
}
```

#### Структура страницы
- **Одноколоночный макет**: все опросы в одной колонке для лучшей читаемости
- **Адаптивный дизайн**: корректное отображение на всех устройствах
- **Карточный интерфейс**: каждый опрос - отдельная карточка с hover-эффектами

### 3. Основные компоненты

#### Заголовок страницы
- Название страницы "Мои опросы"
- Кнопка "Создать опрос" с градиентом и shadow-эффектом
- Адаптивное расположение (вертикальное на мобильных)

#### Фильтры/Табы
- **Три вкладки**: Все / Активные / Завершенные
- Счетчики в скобках для каждой вкладки
- Активная вкладка выделена синим цветом
- Плавные переходы при переключении

#### Карточки опросов
Каждая карточка содержит:

1. **Верхний градиентный бордер**:
   - Зеленый градиент для активных опросов
   - Серый градиент для завершенных опросов

2. **Статус**:
   - Бейдж с иконкой и анимацией пульса для активных
   - Статический бейдж для завершенных
   - Цветовая кодировка (зеленый/серый)

3. **Название опроса**:
   - Крупный, жирный шрифт
   - Hover-эффект с изменением цвета на primary
   - Ограничение в 2 строки с ellipsis

4. **Описание** (опционально):
   - Серый цвет текста
   - Ограничение в 2 строки
   - Скрытие при отсутствии

5. **Метаинформация**:
   - **Дедлайн**: 
     - Иконка часов
     - Оставшееся время для активных
     - Дата завершения для завершенных
     - Красный цвет если < 24 часов
   - **Количество голосов**:
     - Иконка голосования
     - Число голосов с правильным склонением

6. **Кнопки действий**:
   - **Результаты**: полноразмерная кнопка с иконкой
   - **QR-код**: компактная квадратная кнопка
   - Hover-эффекты с изменением цвета

#### Пустое состояние
- Большая иконка опроса (80px)
- Заголовок с объяснением
- Описание в зависимости от выбранной вкладки
- Кнопка "Создать первый опрос" (только для вкладки "Все")

### 4. Интерактивные элементы

#### Анимации и эффекты
- **Карточки**:
  - Плавный hover с увеличением тени
  - Transition-duration: 300ms
  - Transform для легкого поднятия
  
- **Кнопки**:
  - Active:scale-95 для тактильной обратной связи
  - Плавное изменение цвета при hover
  - Shadow-эффекты для primary кнопок

- **Статус активных опросов**:
  - Анимация пульса для точки в бейдже
  - Привлекает внимание к активным опросам

#### Адаптивность
- **Mobile** (< 640px):
  - Вертикальное расположение заголовка и кнопки
  - Одна колонка карточек
  - Компактные отступы

- **Tablet** (640px - 1024px):
  - Горизонтальное расположение элементов
  - Одна колонка карточек

- **Desktop** (> 1024px):
  - Максимальная ширина 1280px (5xl)
  - Одна колонка для лучшей читаемости

### Логика времени

```javascript
const getTimeRemaining = (endDate) => {
  const now = new Date();
  const end = new Date(endDate);
  const diff = end - now;

  if (diff <= 0) return { text: 'Завершен', isUrgent: false };

  const days = Math.floor(diff / (1000 * 60 * 60 * 24));
  const hours = Math.floor((diff % (1000 * 60 * 60 * 24)) / (1000 * 60 * 60));

  const isUrgent = days === 0 && hours < 24; // Срочно если < 24 часов

  if (days > 0) {
    return { text: `${days} дн. ${hours} ч.`, isUrgent };
  }
  return { text: `${hours} ч.`, isUrgent };
};
```

## Сравнение: До и После

### До (Material-UI)
- Сетка 3 колонки на desktop
- Компактные карточки с фиксированной высотой
- Material Design компоненты
- Стандартные MUI тени и эффекты

### После (Tailwind CSS)
- Одна колонка на всех устройствах
- Просторные карточки с flex-высотой
- Кастомные градиенты и анимации
- Современный, чистый дизайн
- Лучшая читаемость и UX

## Преимущества нового дизайна

1. **Современный внешний вид**: чистый, минималистичный дизайн
2. **Лучшая читаемость**: больше пространства, одна колонка
3. **Визуальная иерархия**: четкое разделение элементов
4. **Интерактивность**: плавные анимации и hover-эффекты
5. **Гибкость**: легко модифицировать утилитарные классы
6. **Производительность**: меньший размер CSS bundle
7. **Срочность**: визуальная индикация срочных дедлайнов
